Variables

In Python Programming Language, there are three types of variables. The variable types used in Python are:

  • String
  • Integer
  • Float
  1. String

    2. In Python, any text - whether it be a word or a sentence or a complete paragraph - is a string.

  2. Integer

    3. In Python, a number without a decimal point in between is an integer.

  3. Float

    4. A number with decimal point in between is a float.

Function

Function is the main feature of a modular programming language. A program can be divded into multiple different functions where each function has only one job.

A function can be defined by following syntax:

def function_name(parameters):

__code here__

return variable

Syntax for calling function is:

variable = function_name(arguments)
Class

Python is an Object Oriented Programming Language. So, we use classes. A class is used if we have to define any real world object.

A class can be define with the following syntax:

class ClassName:

def __init__(self, other parameters):

__code here__


def method_name(parameters):

__code here__
return variable

Syntax for initializing a class:

class_variable = class_name(arguments)
variable = class_variable.method_name(arguments)
Module

Python is a modular programming language. This means that we can break a single program into various modules. We can also use the same modules in many different programs.

A module is a Python file which has only functions and classes. Modules are also saved in .py file.

To use a module in our program, we need to import the module in our main file.

from module_file_name import function_name
